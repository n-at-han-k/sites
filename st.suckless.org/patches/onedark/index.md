onedark
=======

Description
-----------
[Atom's onedark theme for terminals](https://github.com/nathanbuchar/atom-one-dark-terminal) mimics the One Dark theme in Atom. It is very popular and serves as the basis for the `Doom-One` 
theme in Doom Emacs.
