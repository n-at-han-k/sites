paste\_controls
===============

Description
-----------
Replaces most [C0 control codes](https://www.ascii-code.com/characters/control-characters)
with spaces when pasting, except for HT and CR.

This can add some security when pasting untrusted buffers, for example
[commands from a browser](https://thejh.net/misc/website-terminal-copy-paste).
